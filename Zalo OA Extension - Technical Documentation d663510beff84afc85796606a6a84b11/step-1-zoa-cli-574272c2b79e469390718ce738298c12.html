[Step 1] ZOA-CLI

1. Cài đặt zoa-cli

yarn: yarn global add zoa-cli

npm: npm i -g zoa-cli

2. Khởi tạo dự án

  • Kiểm tra phiên bản đã cài zoa --version

  • Chuyển vào thư mục project

Chạy lệnh zoa init để khởi tạo project và nhập Zalo App ID

  • Chọn Login via QR code with Zalo App để xác thực tài khoản và lấy access token

  • Chọn Create a new ZOA Extension project và điền thông tin liên quan để tạo project mới

  • Mở project bằng IDE và tiến hành coding

  • Lưu lại thông tin thay đổi và chạy lệnh zoa build

  • Chạy lệnh zoa deploy để sync code lên hệ thống quản lý version và kiểm tra kết quả tại trang [Step 2] Trang quản lý phiên bản
